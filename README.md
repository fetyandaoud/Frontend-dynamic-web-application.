# Posts, Profiles and Comments Web Application

This project is a frontend web application built with HTML, CSS, and JavaScript.  
The application fetches posts, comments, and user profiles from external APIs and displays them dynamically.

## Features
- Responsive navigation bar (Home, Posts, Contact)
- Dynamic posts fetched from an API
- Comments displayed for each post
- User profile modal when clicking a username
- Pagination or infinite scrolling
- Contact form with input validation
- Responsive design using CSS Flexbox/Grid

## Technologies
- HTML
- CSS
- JavaScript
- Fetch API
- Async/Await

## APIs Used
- https://dummyjson.com/posts
- https://dummyjson.com/comments
- https://dummyjson.com/users
